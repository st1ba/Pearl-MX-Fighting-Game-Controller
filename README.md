# 🕹️ Pearl-MX Fighting Game Controller

Custom fighting game controller (leverless / arcade-style) built for precision inputs, low latency, and full customization.

---

## Overview

The **Pearl-MX Fighting Game Controller** is a custom-built controller designed for fighting games (Street Fighter, Tekken, Guilty Gear, etc.).  

## 🧠 Features

- Mechanical switch-based buttons (MX style)
- hitbox/leverless layout
- USB-C connection
